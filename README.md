# reportonlie_os
院内报告在线浏览系统

## 基于django
共分为浏览模块（basehtml）、提取模块（doc2mysql)模块、存储模块（upload）、静态文件模块（static)、以及media模块
此程序可以使没有并入LIS的报告方便快捷的实现院内浏览,因每个报告样式不同，可根据本系统模板、按照自己医院报告的情况自行开发
